### Instalación

Ingresar a cada proyecto y ejecutar

```bash
pip install -r requirements.txt
```

```bash
cd flask && python app.py
```
Probar Flask: http://127.0.0.1:5000/

```bash
cd django_app && python manage.py runserver
```
Probar Django: http://127.0.0.1:8000/culqi/
